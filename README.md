# rustrace
Raytracer in Rust rougly following [Raytracing in One Weekend](https://raytracing.github.io/).

![](final.png)

## Features
- Spheres only right now.
- Materials:
    - Lambertian (diffuse),
    - Dielectric,
    - Metal.
- Anti-Aliasing:
    - Grid,
    - Random.
- Defocus Blur. 
- Parallelised using Rayon.
